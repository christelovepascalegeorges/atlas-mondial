# Atlas Mondial Interactif


## Description

Application web permettant de rechercher un pays et d'afficher ses informations principales grâce à l'API REST Countries.


## Technologies utilisées

- HTML5
- CSS3
- JavaScript ES6
- API REST Countries


## Fonctionnalités

- Recherche d'un pays
- Affichage du drapeau
- Nom du pays
- Capitale
- Population
- Région
- Monnaie
- Langues


## API utilisée

https://restcountries.com/v3.1/name/{nom_du_pays}
